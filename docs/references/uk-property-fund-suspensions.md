## Summary

*[To be populated]* — UK open-ended property funds (M&G, Aviva, Standard Life, Janus Henderson, Aberdeen) repeatedly suspended redemptions when stressed: 2016 (post-Brexit referendum, ~£18B suspended), 2019 (Woodford-driven trust crisis), 2020 (COVID). The structural mismatch — daily-dealing share class + illiquid underlying asset — became a regulatory case study. FCA proposed mandatory notice periods (90–180 days) for daily-dealing property funds (PS19/2 consultation).

## Relevance to MGV

*[To be populated]* — **The clearest precedent for the structural problem MGV solves.** UK property funds suspended because they promised daily liquidity on illiquid assets. MGV explicitly does NOT promise that — the weekly exit cap + queue is the architectural answer to this exact mismatch. Worth citing in §7 to frame the design intent.

Specific lessons to extract:
