## Relevance to MGV

*[To be populated]* — **Stellar precedent.** FOBXX runs on Stellar (among others) — the only tier-1 US-regulated tokenized fund that uses Stellar's transfer layer. Key takeaways:

- Constant-NAV model with dividend-distribution vs MGV's appreciating-NAV — tradeoff analysis worth surfacing
- Recordkeeping via Stellar's transfer layer rather than smart contract — different paradigm from MGV's Soroban contract-as-share-ledger
- Reserve composition + attestation pattern
- Regulatory path (40-Act fund tokenized via blockchain recordkeeping) — useful as a contrast for the MGV regulatory framing (which is NOT a 40-Act fund)

This is also useful as evidence that Stellar can support institutional-tier tokenized fund flows — a non-trivial trust signal for auditors evaluating MGV's chain choice.
