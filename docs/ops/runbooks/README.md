# Operations Runbooks

Operator runbooks describe the actions to take during production incidents or
scheduled protocol operations. They are intentionally procedural and should be
usable by an on-call responder without reading the whole architecture folder.

| Runbook | Trigger summary |
| --- | --- |
| [Pause / unpause](./pause-unpause.md) | Use the Fund kill-switch when the operator hot path must stop while investor recovery paths remain available. |
